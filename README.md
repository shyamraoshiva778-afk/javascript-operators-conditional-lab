# JavaScript Operators & Conditional Lab

A complete vanilla JavaScript lab with 12 interactive projects covering operators, conditional statements, exception handling, BOM (Browser Object Model), and math logic.

## Projects Overview

| # | Project | Concepts |
|---|---------|----------|
| Q1 | Remove Duplicates | String manipulation, objects as hash maps, for loop |
| Q2 | Character Frequency | Object counting, for...in loop |
| Q3 | First Non-Repeating | Two-pass algorithm, string traversal |
| Q4 | String Compression | Run-length encoding, conditional logic |
| Q5 | Exception Handling | try...catch...finally, error handling |
| Q6 | Nested Exception | Multi-level try...catch, error propagation |
| Q7 | Page Redirect | BOM (window.location), navigation |
| Q8 | Digital Clock | setInterval, Date object, DOM updates |
| Q9 | Prime Numbers | Math logic, loops, conditional checking |
| Q10 | Armstrong Number | Mathematical validation, power operations |
| Q11 | Password Generator | Random generation, array methods |
| Q12 | Login Attempt System | Counter logic, lockout handling |

## Running the Projects

### Run Individual Projects
Navigate to any project folder (e.g., `Q1-Remove-Duplicates/`) and open `index.html` in your browser.

### Run Landing Page
Open `index.html` in the root directory to access all projects from a unified interface.

## Learning Topics Covered

- **Operators**: Arithmetic, comparison, logical, bitwise
- **Conditional Statements**: if...else, switch, ternary operator
- **Exception Handling**: try, catch, finally, throw
- **BOM**: window, location, navigator, setInterval
- **Math Logic**: Prime numbers, Armstrong numbers, algorithms
- **String Operations**: Manipulation, compression, frequency analysis
