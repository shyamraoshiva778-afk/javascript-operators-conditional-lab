# Q7 — BOM: Page Redirect System

## Problem Statement
Ask user confirmation before redirecting. Use BOM APIs.

## Concepts Used
- `window.confirm()`
- `window.location.href`
- `navigator.userAgent`
- `screen.width / screen.height`

## How to Run
1. Open `index.html`
2. Click a site card
3. Confirm or cancel redirect
