# Q8 — BOM: Digital Clock

## Problem Statement
Display a live digital clock that updates every second using `setInterval()`.

## Concepts Used
- `setInterval()` / `clearInterval()`
- `Date` object
- BOM — `Intl.DateTimeFormat`
- DOM manipulation

## How to Run
1. Open `index.html`
2. Clock starts automatically
3. Stop / Start using button
